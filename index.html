One-time setup — takes 3 minutes. After this, you and your wife share live data automatically.
1
Go to sheets.google.com and create a new blank spreadsheet. Copy its URL — it looks like:
docs.google.com/spreadsheets/d/SPREADSHEET_ID/edit
Paste the full URL or just the ID below.
2
In that spreadsheet, click Share (top right) → set "Anyone with the link" → Editor. This lets both phones read and write.
3
Go to script.google.com → New Project → paste the script below → click Deploy → New deployment → Web app → Execute as Me, access Anyone → Deploy → copy the Web App URL.
Please fill in both the Spreadsheet ID and Web App URL.
